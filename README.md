Esse é um projeto simples de uma página de login feita por mim para praticar HTML, CSS e JavaScript.

Ela possui:
- Fundo com imagem
- Efeito de blur
- Campo de senha com botão de visualizar
- Animações e efeitos nos botões
- Design responsivo

Fiz esse projeto para treinar front-end e melhorar minhas habilidades na criação de interfaces.
